# Gandhinagar Weather and AQI Data Analysis

This project performs basic data analysis using Python on weather and air quality data of Gandhinagar for the last 10 days.

## Features
- Calculate Average Temperature
- Calculate Median Temperature
- Calculate Average Humidity
- Calculate Median Humidty
- Calculate Average AQI
- Calculate Median AQI
